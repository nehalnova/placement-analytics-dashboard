# Placement Intelligence Dashboard

An end-to-end Data Analytics project that simulates a university placement season using synthetic data and analyzes placement trends through SQL and Power BI.

---

##  Project Overview

This project models the complete placement lifecycle:

Student
→ Application
→ Resume Shortlisting
→ Online Assessment
→ Interviews
→ Offer
→ Acceptance

The objective is to generate actionable insights that can help a university placement cell improve placement performance and recruiter engagement.
